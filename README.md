여행 숙박 사이트 
- 방방곡곡슝슝 (BANGKOK)
- 항공권 / 호텔 / 여행 계획 / 여행 정보 등 다양한 서비스를 제공하는 프로젝트 기획
- 팀원 : 7명 (함수진 - 회원관리)
- 개발기간 : 24.09.11 ~ 26 
- 로그인/로그아웃 , 회원가입, 이메일찾기, 비밀번호찾기(SMTP), 마이페이지(crud - 탈퇴는 update형식), 관리자 회원 리스트(list,view),구글&카카오 RestAPI 사용
